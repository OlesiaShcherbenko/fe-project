## Installation

1. Clone this repository
2. Install NPM packages
    
    For mac, linux and Wwindows:
    ```shell
    npm install
    ```

    For Windows (if any errors):
    ```shell
    npm install --msvs_version=2013
    ```
3. Run development server
    ```shell
        npm run dev
    ```

4. Build for production
    ```shell
        npm run build
    ```
